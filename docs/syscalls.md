# AURUX System Calls (API)

Для вызова системных функций из пользовательского пространства (Ring 3) используется программное прерывание `int 0x80`.

Аргументы передаются через регистры процессора в следующем порядке:
1. `eax` — Номер системного вызова
2. `ebx` — Аргумент 1
3. `ecx` — Аргумент 2
4. `edx` — Аргумент 3
5. `esi` — Аргумент 4

Результат возвращается в регистре `eax`. Значение `-1` или большие отрицательные числа обычно означают ошибку.

## Список поддерживаемых вызовов

| Номер (eax) | Имя | Аргументы | Описание |
| :--- | :--- | :--- | :--- |
| `1` | **print** | `ebx` = `const char* str` | Выводит строку на экран через VGA. |
| `2` | **exit** | `ebx` = `int status` | Завершает текущий процесс и возвращает управление планировщику. |
| `3` | **read** | `ebx` = `int fd`<br>`ecx` = `char* buf`<br>`edx` = `size_t count` | Читает данные из файла, пайпа или клавиатуры (fd=0). Возвращает количество прочитанных байт. |
| `4` | **elf_load** | `ebx` = `const char* filename` | (Устарело) Загружает ELF без аргументов. |
| `5` | **wait** | `ebx` = `int pid` | Приостанавливает текущий процесс до завершения процесса с указанным `pid`. |
| `6` | **open** | `ebx` = `const char* filename` | Открывает файл и возвращает дескриптор файла (`fd`). При ошибке возвращает `-1`. |
| `7` | **write** | `ebx` = `int fd`<br>`ecx` = `const char* buf`<br>`edx` = `size_t count` | Записывает данные в файл или пайп по дескриптору. |
| `8` | **close** | `ebx` = `int fd` | Закрывает открытый файл или конец пайпа. |
| `9` | **seek** | `ebx` = `int fd`<br>`ecx` = `int offset` | Устанавливает смещение (курсор) для чтения/записи файла. |
| `10` | **mkdir** | `ebx` = `const char* filename` | Создает новую директорию (FAT32). |
| `11` | **list_dir**| `ebx` = `const char* filename`<br>`ecx` = `char* buf`<br>`edx` = `size_t size`<br>`esi` = `int detailed` | Получает список содержимого директории. Если `detailed = 1`, выводит с размерами и датами. |
| `12` | **unlink** | `ebx` = `const char* filename` | Удаляет файл (FAT32). |
| `13` | **chdir** | `ebx` = `const char* filename` | Меняет текущую рабочую директорию (CWD) для процесса. |
| `14` | **getcwd** | `ebx` = `char* buf`<br>`ecx` = `size_t size` | Получает текущую рабочую директорию. |
| `15` | **create_file**| `ebx` = `const char* filename` | Создает пустой файл. |
| `16` | **gettime** | `ebx` = `int* time_arr` | Заполняет массив из 6 чисел (год, месяц, день, часы, минуты, секунды) текущим временем с RTC. |
| `17` | **pipe** | `ebx` = `int fd[2]` | Создает анонимный канал (pipe) для межпроцессного взаимодействия. `fd[0]` — чтение, `fd[1]` — запись. |
| `18` | **fork** | Нет | Создает точную копию текущего процесса. Возвращает `0` дочернему процессу и PID ребенка — родительскому. |
| `19` | **exec** | `ebx` = `const char* path`<br>`ecx` = `const char* args` | Заменяет образ текущего процесса новой программой ELF и передает ей аргументы командной строки. |
| `20` | **sbrk** | `ebx` = `unsigned int end_addr` | Изменяет размер кучи процесса (выделяет виртуальную память). Если `0`, возвращает конец кучи. |
| `21` | **dup2** | `ebx` = `int oldfd`<br>`ecx` = `int newfd` | Дублирует файловый дескриптор в `newfd`. Используется для перенаправления ввода-вывода (>, <). |
| `22` | **sleep** | `ebx` = `unsigned int ms` | Усыпляет текущий процесс на указанное количество миллисекунд (без блокировки ОС). |
| `23` | **kill** | `ebx` = `int pid`<br>`ecx` = `int signal` | Посылает сигнал (например, `SIGINT = 2`) процессу с указанным `pid` (завершает его). |
| `24` | **get_processes** | `ebx` = `void* buffer`<br>`ecx` = `int max_count` | Возвращает список активных процессов (PID, статус, имя). |
| `25` | **getuid** | Нет | Возвращает идентификатор текущего пользователя (UID) для процесса. |
| `26` | **setuid** | `ebx` = `int uid` | Устанавливает идентификатор пользователя (UID). |
| `27` | **getgid** | Нет | Возвращает идентификатор текущей группы (GID) для процесса. |
| `28` | **setgid** | `ebx` = `int gid` | Устанавливает идентификатор группы (GID). |
| `29` | **chown** | `ebx` = `const char* path`<br>`ecx` = `int uid`<br>`edx` = `int gid` | Изменяет владельца и группу файла. |
| `30` | **chmod** | `ebx` = `const char* path`<br>`ecx` = `int mode` | Изменяет права доступа (режим) файла. |
| `31` | **umask** | `ebx` = `int mask` | Устанавливает маску прав для новых файлов. Возвращает старую маску. |
| `32` | **stat** | `ebx` = `const char* path`<br>`ecx` = `struct fs_stat* st` | Получает метаданные файла (права, владелец, размер). |
| `33` | **geteuid** | Нет | Возвращает эффективный идентификатор пользователя (euid) для процесса. |

### Безопасность
Ядро AURUX проверяет все передаваемые указатели (например, `const char* str`). Если пользовательская программа попытается передать указатель на память ядра (ниже `0x01000000`) или на невыделенную страницу, процесс будет немедленно завершен с ошибкой **SIGSEGV** (Segmentation Fault).
