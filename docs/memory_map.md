# AURUX Memory Map

Архитектура управления памятью разделена на физическую и виртуальную (через механизм Paging). 
В AURUX реализована строгая изоляция между ядром (Ring 0) и пользовательскими программами (Ring 3).

## Физическая память (Physical Memory)

| Диапазон адресов | Описание |
| :--- | :--- |
| `0x00000000 - 0x000FFFFF` | Первые 1 МБ. Зарезервировано BIOS, память видеокарты VGA (`0xB8000`), векторы прерываний реального режима (не используются после перехода в защищенный режим). |
| `0x00100000 - 0x00FFFFFF` | Пространство ядра. Начинается с `0x100000` (1 МБ). Здесь загружено само ядро, его код (`.text`), данные (`.data`) и куча (kmalloc). |
| `0x01000000 - Конец ОЗУ` | Пространство для пользовательских процессов (от 16 МБ и выше). Физические страницы для программ Ring 3 выделяются отсюда. |

## Виртуальная память (Virtual Memory)

Благодаря страничной адресации (Paging) и таблицам страниц (Page Directory / Page Tables), каждый процесс получает собственное изолированное виртуальное адресное пространство.

| Диапазон адресов | Права доступа | Описание |
| :--- | :--- | :--- |
| `0x00000000 - 0x07FFFFFF` | **Ring 0** (Ядро) | Identity-mapped (виртуальный адрес = физическому). Недоступно для чтения и записи из Ring 3. Если пользовательская программа попытается обратиться сюда, возникнет Page Fault. |
| `0x08048000 - ...` | **Ring 3** (Пользователь) | Базовый адрес для загрузки ELF программ. Каждая программа компилируется с учетом того, что её стартовый адрес — `0x08048000`. Память мапится на физические страницы, начиная с `0x01000000`. |
| `... - 0xBFFFFFFF` | **Ring 3** (Пользователь) | Доступная память для стека и кучи процессов. |

### Механизм аллокации
- **Физический аллокатор (PMM)**: Использует растровую карту (bitmap). Размер выделяемого блока всегда равен 1 странице (`4096` байт).
- **Виртуальный аллокатор ядра (KMM)**: Алгоритм First-Fit с поддержкой фрагментации и слияния свободных блоков. Используется ядром для выделения небольших структур данных (например, структур задач `task_t` или буферов файловой системы).
